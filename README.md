# bootstrap-custom
> A lightweight, customizable Bootstrap 4 partial import package.

## description
- full: 全量的 bootstrap
- lite: 与 tailwindcss 兼容的 bootstrap
- slim: 精简的 bootstrap - 没有 buttons 等 components
